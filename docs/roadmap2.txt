1. Architektura Technologiczna -- Z czego sklada sie plugin

Fraction Guild Clans v2.0 to samowystarczalny plugin na serwery Paper/Spigot/Purpur (Minecraft 1.8Ł1.21.11). Nie wymaga zadnych zewnetrznych zaleznosci.

1.1. Stack Technologiczny

Komponent Jezyk Build Tool Baza danych Pool polaczen Wiadomosci GUI System modulow

Technologia Java 17+ Maven lub Gradle SQLite (domyslnie), MySQL HikariCP MiniMessage + legacy &k Wlasny system oparty na Bukkit Inventory API Wlasny loader z sortowaniem topologicznym

Uzasadnienie Kompatybilnosc z Paper API 1.8Ł1.21 Standard ekosystemu Bukkit Wbudowane, bez zewnetrznych pluginow Wydajnosc przy duzej liczbie graczy Wsparcie nowych i starych formatow Pelna konfiguracja bez ograniczen Zaleznosci miedzy modulami

1.2. Filozofia "Zero Zaleznosci"
NIE uzywamy:
- Vault -- zastepujemy wlasnym EcoCore (lub trybem itemowym) - WorldGuard/WorldEdit -- wlasny system ochrony cuboidow - EssentialsX/CMI -- wlasny system ekonomiczny - Zadnych pluginow ekonomicznych -- EcoCore jest czescia ekosystemu
Opcjonalnie integrujemy:
- PlaceholderAPI -- rejestracja placeholderow %fgc_* - EcoCore -- zaawansowana ekonomia (plugin tego samego autora)
