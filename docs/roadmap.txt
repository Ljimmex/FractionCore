1. Architektura Technologiczna -- Z czego sklada sie plugin

Fraction Guild Clans v2.0 to samowystarczalny plugin na serwery Paper/Spigot/Purpur (Minecraft 1.8ş1.21.11).
Nie wymaga zadnych zewnetrznych zaleznosci.

1.1. Stack Technologiczny

   Komponent       Technologia                                     Uzasadnienie
   Jezyk           Java 17+                                        Kompatybilnosc z Paper API 1.8ş1.21
   Build Tool      Maven lub Gradle                                Standard ekosystemu Bukkit
   Baza danych     SQLite (domyslnie), MySQL                       Wbudowane, bez zewnetrznych pluginow
   Pool polaczen   HikariCP                                        Wydajnosc przy duzej liczbie graczy
   Wiadomosci      MiniMessage + legacy &k                         Wsparcie nowych i starych formatow
   GUI             Wlasny system oparty na Bukkit Inventory API    Pelna konfiguracja bez ograniczen
   System modulow  Wlasny loader z sortowaniem topologicznym       Zaleznosci miedzy modulami

1.2. Filozofia "Zero Zaleznosci"

NIE uzywamy:

- Vault -- zastepujemy wlasnym EcoCore (lub trybem itemowym)
- WorldGuard/WorldEdit -- wlasny system ochrony cuboidow
- EssentialsX/CMI -- wlasny system ekonomiczny
- Zadnych pluginow ekonomicznych -- EcoCore jest czescia ekosystemu

Opcjonalnie integrujemy:

- PlaceholderAPI -- rejestracja placeholderow %fgc_*
- EcoCore -- zaawansowana ekonomia (plugin tego samego autora)
